Tutorials
=========

.. image:: ../_static/structure_property.001.jpeg
   :width: 100%
   :align: center

The tutorials include introductions to concepts of Allen-Feldman diffusivity, Bond-Network Entropy and Disorder Linewidth.

Throughout the tutorials we will use structures that were created and used in a research environment, and discussed in scientific publications. Specifically:

- the structures of irradiated graphite used in diffusivity and disorder linewidth tutorials can be found at the following link: https://doi.org/10.24435/materialscloud:jm-cg. 
  If you use these structures of irradiated graphite in your research, you should cite both :cite:t:`tut-iwanowski_bond-network_2025` and :cite:t:`tut-farbos_time-dependent_2017`.
- the structure of silica glass used in the bond-network entropy tutorial can be found at the following link: https://doi.org/10.24435/materialscloud:3k-v7. 
  If you use this structure in your research, you should cite both :cite:t:`tut-simoncelli_temperature-invariant_2025` and :cite:t:`tut-erhard_machine-learned_2022`.
